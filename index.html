에어테이블 연결 설정

이 페이지는 에어테이블 마스터 데이터 베이스에서 데이터를 직접 불러옵니다. 최초 1회만 개인 액세스 토큰을 입력하면 이 브라우저에 저장됩니다.

  2. Scopes에 data.records:read 추가
  3. Access에 마스터 데이터 베이스 추가 후 토큰 생성
  4. 생성된 토큰(pat…)을 아래에 붙여넣기
